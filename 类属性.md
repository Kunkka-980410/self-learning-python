## 类属性
是属于类本身的属性，被所有类的实例所共享  

**类属性 vs 实例属性**
```python
class Player:
    # 类属性 - 属于类本身，所有实例共享
    game = "王者荣耀"
    total_players = 0
    
    def __init__(self, name):
        # 实例属性 - 属于各个实例对象
        self.name = name
        self.level = 1
        # 修改类属性（通过类名访问）
        Player.total_players += 1

# 使用示例
print(Player.game)  # 通过类名访问类属性：王者荣耀
print(Player.total_players)  # 0

player1 = Player("张三")
player2 = Player("李四")

print(Player.total_players)  # 2 - 类属性被所有实例共享和修改
```
**类属性的特点**
1. 共享性
```python
class Config:
    # 类属性 - 配置信息，所有实例共享
    version = "1.0.0"
    max_level = 100
    supported_platforms = ["Windows", "macOS", "Linux"]

# 所有实例都共享相同的类属性
config1 = Config()
config2 = Config()

print(config1.version)  # 1.0.0
print(config2.version)  # 1.0.0

# 通过类名修改会影响所有实例
Config.version = "2.0.0"
print(config1.version)  # 2.0.0
print(config2.version)  # 2.0.0
```
2. 访问方式
```python
class Counter:
    count = 0  # 类属性
    
    def __init__(self):
        # 两种访问方式：
        Counter.count += 1  # 推荐：通过类名访问
        # 或者 self.__class__.count += 1
        self.id = Counter.count

# 使用
c1 = Counter()
c2 = Counter()
c3 = Counter()

print(Counter.count)  # 3
print(c1.id, c2.id, c3.id)  # 1 2 3
```

class MathConstants:

2. 计数器
```python
class User:
    user_count = 0
    online_users = []
    
    def __init__(self, username):
        self.username = username
        User.user_count += 1
        User.online_users.append(username)
    
    def logout(self):
        User.online_users.remove(self.username)

# 使用
user1 = User("Alice")
user2 = User("Bob")
user3 = User("Charlie")

print(f"总用户数: {User.user_count}")  # 总用户数: 3
print(f"在线用户: {User.online_users}")  # 在线用户: ['Alice', 'Bob', 'Charlie']

user2.logout()
print(f"在线用户: {User.online_users}")  # 在线用户: ['Alice', 'Charlie']
```
3. 默认值设置
```python
class Car:
    # 类属性 - 默认配置
    wheels = 4
    fuel_type = "汽油"
    max_speed = 200
    
    def __init__(self, brand, color, max_speed=None):
        self.brand = brand
        self.color = color
        # 实例属性可以覆盖类属性
        self.max_speed = max_speed or Car.max_speed
    
    def info(self):
        return f"{self.color}的{self.brand}，{self.wheels}个轮子，最高时速{self.max_speed}km/h"

# 使用
car1 = Car("宝马", "黑色")
car2 = Car("奔驰", "白色", 250)

print(car1.info())  # 黑色的宝马，4个轮子，最高时速200km/h
print(car2.info())  # 白色的奔驰，4个轮子，最高时速250km/h
```
**重要注意事项**
1. 修改类属性的陷阱
```python
class Test:
    shared_list = []  # 类属性是可变对象
    
    def __init__(self, value):
        self.shared_list.append(value)  # 这会修改所有实例共享的列表

t1 = Test(1)
t2 = Test(2)
t3 = Test(3)

print(Test.shared_list)  # [1, 2, 3] - 所有实例共享同一个列表！
```
2. 正确的做法
```python
class Test:
    shared_list = None  # 先定义为None
    
    def __init__(self, value):
        if Test.shared_list is None:
            Test.shared_list = []  # 第一次使用时初始化
        self.shared_list.append(value)

# 或者使用实例属性
class BetterTest:
    def __init__(self, value):
        self.my_list = []  # 每个实例有自己的列表
        self.my_list.append(value)
```
3. 类属性与实例属性同名
```python
class Example:
    value = "类属性"  # 类属性
    
    def __init__(self):
        self.value = "实例属性"  # 实例属性

obj = Example()
print(obj.value)        # 实例属性 - 实例属性优先
print(Example.value)    # 类属性 - 通过类名访问
```

**总结**
### 类属性的核心特点：

✅ 共享性：所有实例共享同一个类属性

✅ 通过类名访问：ClassName.attribute_name

✅ 节省内存：相同数据不需要在每个实例中存储

✅ 适合存储：常量、配置信息、计数器等共享数据

**使用场景**：

应用程序的配置信息

统计实例数量

定义常量

共享的缓存数据

默认值设置
